# Blinkit Sales Analysis – Excel Project

## Overview
This project analyzes Blinkit sales data using Microsoft Excel to identify
sales trends, outlet performance, product contribution, and customer preferences.

The project includes both analytical pivot-based analysis and a business-ready dashboard.

## Tools & Techniques
- Microsoft Excel
- Pivot Tables & Pivot Charts
- Slicers (Outlet Size, Location, Item Type)
- KPIs & Dashboard Design

## Key KPIs
- Total Sales: ₹1.2M
- Average Sales: ₹141
- Number of Items: 8523
- Average Rating: 4.0

## Key Insights
- Tier 3 outlets generate the highest total sales.
- Medium-sized outlets outperform small and large outlets.
- Low-fat items contribute more sales than regular items.
- Fruits & Vegetables and Snack Foods are top-performing categories.
- Peak sales observed around 2018.

## Project Structure
- data/: Raw sales dataset
- analysis/: Pivot-based analytical workbook
- dashboard/: Interactive Excel dashboard
- screenshots/: Dashboard and analysis previews
